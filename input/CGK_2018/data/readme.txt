This folder provides replication files for 
"How do firms form their expectations? New survey evidence"
by Coibion, Gorodnichenko, and Kumar

STATA do files are in subfolder "do_files". 
The name of a do-file indicates which figure/table the file produces.
runme.do file produces all tables and figures reported in the paper. 

Subfolder "output" includes files necessary for replication. 
The file with micro-level survey data is master_file_public_release.dta. 

Output is stored in subfolders "figures" and "output".
